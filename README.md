# 🚀 GitHub Trend Analysis Dashboard

## 🔗 在线访问
👉 https://app-trend-analysis.streamlit.app/

## 📊 项目简介
基于 GitHub API 的热门仓库趋势分析系统，
通过 GitHub Actions 实现每日自动更新，
并构建热度评分模型（Hot Score）。

## 🧠 技术栈
- Python
- Pandas
- Plotly
- Streamlit
- GitHub Actions

## ⚙️ 核心功能
- 热门仓库排行
- 技术语言分布
- 热度评分模型
- 趋势变化分析

## 🔄 自动化
每日自动抓取 + 更新数据 + 生成分析结果
